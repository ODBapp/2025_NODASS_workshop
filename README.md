# 2025_NODASS_workshop
Lectures and codebase for 2025 NODASS Workshop (2025-07-16 ~ 2025-07-18)

### Overview

   本課程與 Python 程式碼由 ODB 技術員寫作與維持，著重於海洋資料整理，包含地理資訊格式、海洋生物資料標準化、時間序列等，與海洋科學資料分析初探，以海洋熱浪與氣象浮標為基本範例。

### 📁 Directory structure
